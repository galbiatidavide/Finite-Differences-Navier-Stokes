class \mbox{\hyperlink{classeuler__problem}{euler\+\_\+problem}}
\begin{DoxyCompactList}\small\item\em Solves the Euler evolutionary incompressible equations, applying a first order Chorin-\/\+Temam algorithm. It consists of a transport problem for each velocity component and a Poisson problem for the pressure. Current implementation allows Dirichlet bc\textquotesingle{}s only. Boundary conditions are accessed from reference solution, which is a function of space and time. Current implementation DOES NOT ALLOW for Brinkamn penalty method. \end{DoxyCompactList}\end{DoxyCompactItemize}
