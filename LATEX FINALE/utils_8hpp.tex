\doxysubsection{Detailed Description}
Utility functions for grid creation, analytical solutions, and geometric operations. 

This file contains various utility functions used in numerical simulations, including\+:
\begin{DoxyItemize}
\item Functions for generating analytical velocity solutions.
\item Grid creation and setup.
\item Mesh operations such as point-\/in-\/mesh checks and ray-\/triangle intersections.
\item Functions for creating Brinkman masks. 
\end{DoxyItemize}
