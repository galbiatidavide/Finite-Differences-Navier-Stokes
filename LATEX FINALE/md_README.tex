This project implements a solver based on finite differences for the Navier-\/\+Stokes equations. The solver can be used both on a default cubic domain as well as on an input original mesh which exploits the brinkman penalization method.\hypertarget{md_README_autotoc_md1}{}\doxysection{$<$strong$>$\+Requirements$<$/strong$>$}\label{md_README_autotoc_md1}
